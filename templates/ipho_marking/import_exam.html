Importing the subquestion from an exam means:

  • Add and overwrite subquestion meta data.
    Note that this will not delete current subquestion meta data, current subquestions are idenfitied by their name, which could also cause false-matches.
  • Generate default entry for all students (null points).
    Note that points already set will not be reset.

Note that this action can take several minutes. Please be patient and do not reload the page.
